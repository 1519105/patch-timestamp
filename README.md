# patch-timestamp
Add timestamp into camera_pi.py on raspicamsrv

[https://signag.github.io/raspi-cam-srv/4.8/](https://signag.github.io/raspi-cam-srv/4.8/)

Just run:
> ```bash
> curl -sL https://raw.githubusercontent.com/1519105/patch-timestamp/main/patch_timestamp.py -o patch_timestamp.py
> python3 patch_timestamp.py
> ```
